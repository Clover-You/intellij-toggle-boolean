<!-- Plugin description -->
- 在布尔值（即真和假）之间轻松切换。
- 切换选定的布尔值，或仅切换光标下方的布尔值。
- 默认键绑定：alt+b（或设置自定义键）。

## 支持的布尔值列表

- true <-> false
- off <-> on
- yes <-> no
- 1 <-> 0
- enable(d) <-> disable(d)
- before <-> after
- first <-> last

## Action

提供了以下命令

- `ToggleBool` 切换 bool
